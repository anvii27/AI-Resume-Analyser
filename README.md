# AI Resume Analyzer

An AI-powered web application that analyzes resumes against job descriptions using NLP similarity scoring and AI feedback.

## Features
- Upload resume PDF
- Paste job description
- Match percentage score
- AI-based improvement suggestions
- Light/Dark mode UI

## Tech Stack
- React + Vite
- Node.js
- PDF parsing
- NLP similarity
- AI APIs

## How to Run
npm install  
npm run dev
